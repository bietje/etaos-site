Detailed Description

ETA/OS kernel API.

The kernel API supports the following features:

  • Scheduling/Multithreading
  • IRQ management
  • Memory management
  • I/O drivers
  • Timer management

Minimal application

An example of a most minimal application:

#include <etaos/kernel.h>
#include <etaos/types.h>
void main(void)
{
for(;;);
}
