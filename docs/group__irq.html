Detailed Description

Interrupt support.

Support to register interrupt handlers to the kernel. They can be registered either as tradition IRQ handlers or as threaded IRQ's. Please note that threaded IRQ's are slower.

Note
Note, that it is not required to register interrupts with Nut/OS. Actually, this will route interrupts through the kernel, which results in performance degradation.
